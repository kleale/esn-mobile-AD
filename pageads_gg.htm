Branding

  • Width — 1920 pixels
  • Height — 1080 pixels
  • Content width 1150 pixels
  • Max 750 kilobytes
  • Advertising message block height (from topbar to site): 230 pixels
  • The bottom part should turn into a solid color, you need to send a gradient in 6-digit format
Link to archive with branding examples

Fullscreen

The banner is shown over of the content for a certain amount of time by a timer, or until the user closes the banner by clicking the Close button.

HTML format or PNG/JPG images

It is allowed to develop an HTML creative according to the AdFox technical requirements https://yandex.ru/support/adfox-sites/banners/specs/banner-html5.html

  • Width — 900 pixels (100%)
  • Height — 600 pixels (100%)
  • The weight of the archive and all uploaded files — no more than 300 kilobytes
  • The maximum allowed number of characters in the HTML code — 65 000

Banner (Billboard)

Banner fixed at the top of the browser window.

HTML format or GIF/PNG/JPG images.

  • Width – 1250 pixels (100%)
  • Height – 90 pixels
  • Max 750 kilobytes

The client independently implements video playback in the banner itself (calling the player, control elements). Video playback must be initiated by the user (on click). The presence of the "Stop/ Play" buttons is necessarily. The sound (if any) starts strictly on click.

Weight — up to 1 megabyte

The format — MP4
